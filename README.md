# 🚀 Research & Innovation Website (Next.js + TS)

A modern research & innovation site built with **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, **GSAP**, and **Lucide Icons**.  
Includes: News, Events, Image/Video Galleries, Achievements, Partnerships, Press Releases, and an Admin Dashboard with a simple static login.

---

## ✨ Features

- 📰 News & Press Release archive with single article pages
- 📅 Events (grid + single)
- 🏆 Achievements page
- 🤝 Partnerships page
- 🖼️ Image Gallery (albums) & 🎥 Video Gallery (albums or event-based)
- 🔐 Admin Dashboard (static login token)
- 🎬 Video modal (YouTube) with `react-player`
- ⚡ Smooth animations (Framer Motion + GSAP)
- 🧭 Responsive Navbar with login/logout state
- 🖼️ Optimized images via `next/image`

---

## 🧰 Tech Stack

- **Next.js 14** (App Router)
- **React 18** + **TypeScript**
- **Tailwind CSS**
- **Framer Motion** + **GSAP**
- **Lucide-react** (icons)
- **react-player** (dynamic import for SSR)
